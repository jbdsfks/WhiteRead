简介
-----
这是一个之前就想弄的东西，作为一个新手，正好最近又想学习一下Rxjava免得还没入门就要被淘汰，所以写了这么一个聚合阅读软件。

使用到的技术点：

**android原生**
 - recyclerview
 - cardview
**第三方库**
 - butterKnife
 - RxJava
 - gson
 - retrofit
 - okhttp
 - glide
 - fresco
 - Jsoup
 - logger
 - magicindicator
 - photoview
 - convenientbanner
 - xrecyclerview

PS：该项目主要是用于学习Jsoup的Html解析，recycler的灵活使用，fresco和glide的图片加载对比，okhttp的原生应用和与retrofit的结合使用，Rxjava的入门学习和okhttp+retrofit+Rxjava的结合。
